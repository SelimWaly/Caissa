# Chess
UCI Chess Engine written in C++. Currently compiles only for Windows.

## Features

  * UCI protocol
  * Alpha-Beta prunning
  * Principal Variation Search
  * Transposition Table
  * MVV/LVA
  * Killer/history heuristics
  * Iterative Deepening
  * Aspiration Window
  * Late Move Reduction
  * Multi-PV search
  * Null Move Prunning
  * Magic Bitboards
